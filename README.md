# OIBSIP_DataAnalytics_Task2
Customer Segmentation Analysis

## Customer Segmentation Analysis

This project performs **Customer Segmentation Analysis** for an e-commerce business using data analytics techniques. The aim is to divide customers into different groups based on their buying patterns and demographic details. These segments help businesses understand customer behavior and design more effective marketing strategies.

## Dataset

**ifood_df.csv**

The dataset includes information about customers such as demographic details, purchasing habits, and behavioral attributes that can be used to analyze and classify customers into meaningful segments.

## Objectives

* Explore and understand the dataset
* Perform data cleaning and preprocessing
* Study customer purchasing patterns using statistical analysis
* Segment customers using clustering techniques
* Visualize the identified customer groups using graphs and charts
* Generate useful insights for business decision-making

## Methodology

The following steps were followed during the analysis:

1. Data cleaning and preparation of the dataset
2. Selection of relevant features for segmentation
3. Feature scaling using **StandardScaler**
4. Application of **K-Means clustering** to group customers
5. Visualization of customer segments using charts and plots
6. Analysis and interpretation of the resulting customer groups

## Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## Results

The analysis successfully divided customers into several distinct segments based on factors such as income level, spending behavior, and purchasing activity. Each group represents customers with similar characteristics.

## Conclusion

Customer segmentation enables businesses to gain a deeper understanding of their customer base. By identifying different customer groups, companies can personalize marketing campaigns, improve customer engagement, and make better data-driven business decisions.
